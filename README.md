# Rencana Pembelajaran Semester (RPS)  
## Mata Kuliah: Pemrograman Mobile 2 (IF-KK5004)  
**Bobot/Semester:** 3 SKS / Semester 5  
**Prasyarat:** IF-KK4002 Pemrograman Mobile 1  

---

## Deskripsi
Mata kuliah **Pemrograman Mobile 2** merupakan pengembangan aplikasi mobile multi-platform dengan **Flutter** yang dirancang untuk membekali mahasiswa dengan pengetahuan dan keterampilan dalam membangun aplikasi modern di berbagai platform seperti **Android** dan **iOS**.  

Materi mencakup:  
- Bahasa pemrograman **Dart**  
- Arsitektur Flutter  
- Implementasi UI interaktif & responsif (widget dasar & lanjutan)  
- Manajemen state, navigasi, animasi  
- Integrasi API, penyimpanan lokal  
- Pemanfaatan layanan eksternal (Firebase)  
- Pendekatan berbasis proyek (perancangan, pengembangan, pengujian, publikasi)  

---
